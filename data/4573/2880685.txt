Launchpad is a web application and website that allows users to develop and maintain software particularly open source software It is developed and maintained by Canonical Ltd br On July the source code was released publicly under the GNU Affero General Public License br As of June the Launchpad repository hosts more than projects The domain launchpad net attracted million visitors by August according to a Compete com survey br br br Components br It has several parts br br Answers a community support site and knowledge base br Blueprints a system for tracking new features br Bugs a bug tracker that allows bugs to be tracked in multiple contexts e g in an Ubuntu package as an upstream or in remote bug trackers br Code source code hosting with support for the Bazaar and Git version control systems br Translations a site for localising applications into different languages br A significant but less visible component is Soyuz the distribution management portion of Launchpad Launchpad is currently primarily used in the development of Ubuntu an operating system Launchpad uses the FOSS free open source Zope application server br br br Users br Several of Canonical Ltd s own projects use Launchpad for development including Ubuntu and Bazaar Development of Launchpad is itself managed in Launchpad br Other prominent projects using Launchpad for various aspects of managing their development include br br JOSM translations br Linux Mint translations br MariaDB mailing list br OpenStack bug tracking br Pinta bug tracking and translations br Upstart br elementary OS br br br Transition to free software br Launchpad was initially criticized by the Jem Report and other members of the free software community for not being available under a free license such as the GNU GPL despite its aims In response the developers stated that they aimed to eventually release it under a free software license but that it could potentially take years On July Canonical Ltd released Storm the first Launchpad component made available under a free software license br Founder Mark Shuttleworth s responded to this criticism saying we are all actively working on making Launchpad open source adding that the funding the salaries of Launchpad s developers to be higher priority and claiming immediate release would result multiple unfederated instances of Launchpad However this still left some members of the open source movement dissatisfied On July Mark Shuttleworth announced at OSCON that the complete source code would be released within the next twelve months br On December Canonical Ltd released the Launchpad component lazr config and lazr delegates under version of the GNU LGPL br An open API is currently in beta testing which will allow programs to interact with the website Calls for an open API to be released were aided by projects like Leonov that resorted to screen scraping to get data from Launchpad br In December Canonical announced that the source code to the Launchpad website would be released under a free software license by July It was also announced that two large components of Launchpad Soyuz which is responsible for the build system package management and Ubuntu package publishing and Codehosting would not be released under a free software license Later the specific date was changed to a more general timeframe of July August However on July the software was released under the AGPLv a fully free license specifically for web services including the two components Codehosting and Soyuz that were initially planned to remain proprietary
