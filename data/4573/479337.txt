Mozilla Sunbird is a discontinued free and open source cross platform calendar application that was developed by the Mozilla Foundation Sun Microsystems and many volunteers Mozilla Sunbird was described as a cross platform standalone calendar application based on Mozilla s XUL user interface language Announced in July Sunbird was a standalone version of the Mozilla Calendar Project br It was developed as a standalone version of the Lightning calendar and scheduling extension for the Mozilla Thunderbird and SeaMonkey mail clients Development of Sunbird was ended with release beta to focus on development of Mozilla Lightning The latest development version of Sunbird remains b from January and no later version has been announced Unlike Lightning Sunbird no longer receives updates to its time zone database br br br Sun contributions br Sun Microsystems contributed significantly to the Lightning extension project to provide users with a free and open source alternative to Microsoft Office by combining OpenOffice org and Thunderbird Lightning Sun s key focus areas in addition to general bug fixing were calendar views team collaboration features and support for the Sun Java System Calendar Server Since both projects share the same code base any contribution to one is a direct contribution to the other br br br Trademark issues and Iceowl br br Although it is released under a MPL MPL GPL LGPL tri license there are trademark restrictions in place on Mozilla Sunbird which prevent the distribution of modified versions with the Mozilla branding br As a result the Debian project created Iceowl a virtually identical version without the branding restrictions br br br Release history br br br See also br Lightning for Mozilla Thunderbird and SeaMonkey br List of personal information managers br br br br br br External links br Official website br MozillaWiki br The Sunbird development blog br Sunbird Portable by PortableApps com br Linux sunbird installer
