MINCE is a text editor originally created for based microcomputers running the CP M operating system Later versions of MINCE were available for GEMDOS on the Atari ST VAX VMS RSX and various flavors of Unix br Developed in BDS C by Mark of the Unicorn it was bundled with computers from Kaypro and Morrow Designs and the Epson QX It was a subset of Emacs MINCE stands for MINCE Is Not Complete Emacs designed to run within the kB memory limit of bit computers br MINCE used a gap buffer to fit within kB and implemented a very efficient virtual memory system to support multiple buffers and a maximum file size limited only by available disk space br MINCE was a companion product to SCRIBBLE a text formatter based on Scribe This separation of duties into editor plus formatter was common among advanced word processors at that time br Although it was not open source MOTU distributed partial code they deemed most useful for extending the product br In MINCE and SCRIBBLE were sold together along with their source code and the BDS C compiler as a software bundle for US almost US in dollars under the name Amethyst Amethyst was available without the compiler for and MINCE and SCRIBBLE were available alone for In the list price of MINCE was US equivalent to US in br MINCE and SCRIBBLE were later developed into the Perfect Writer and FinalWord word processors FinalWord later became Sprint br An open source project Portable MINCE allows the CP M version of MINCE to run on current operating systems and provides customizations extensions and documentation for the editor br br br br br br External links br Portable MINCE GitHub com Retrieved br The Unicorn Fable Scrounge org Retrieved br The Craft of Text Editing Finseth com Archived from the original on Retrieved br TextEditors Wiki Mince Texteditors org Retrieved br Mince Z TXT Retroarchive org Retrieved br The Ohio State University Newsgroup Query TXT Retroarchive org Retrieved
