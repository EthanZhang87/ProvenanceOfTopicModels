IntelliJ IDEA is an integrated development environment IDE written in Java for developing computer software written in Java Kotlin Groovy and other JVM based languages It is developed by JetBrains formerly known as IntelliJ and is available as an Apache Licensed community edition and in a proprietary commercial edition Both can be used for commercial development br br br History br The first version of IntelliJ IDEA was released in January and was one of the first available Java IDEs with advanced code navigation and code refactoring capabilities integrated br In JetBrains released the source code for IntelliJ IDEA under the open source Apache License JetBrains also began distributing a limited version of IntelliJ IDEA consisting of open source features under the moniker Community Edition The commercial Ultimate Edition provides additional features and remains available for a fee br In a InfoWorld report IntelliJ received the highest test center score out of the four top Java programming tools Eclipse IntelliJ IDEA NetBeans and JDeveloper br In December Google announced version of Android Studio an open source IDE for Android apps based on the open source community edition
