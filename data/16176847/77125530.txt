macOS Sequoia version is the upcoming major release of Apple s macOS operating system The successor to macOS Sonoma it was announced at WWDC on June It is named after Sequoia National Park which is located in the Sierra Nevada mountain range br The first developer beta was released on June The public beta is expected to be released in July br br br Development br br br Announcement br macOS Sequoia was announced by senior vice president of software engineering Craig Federighi at Apple s Worldwide Developers Conference on June br br br Features br macOS Sequoia includes a number of new features and improvements mainly focused on productivity br br iPhone Mirroring a feature to mirror and interact with content from an iPhone as a macOS window has been introduced br The Calculator app has been redesigned to be more similar to that of iOS and iPadOS including rounded buttons departing from the design that has been used since OS X Yosemite br Notes has been updated with the Math Notes feature which can be used to calculate simple equations evaluate expressions and assign variables within the app br Passwords a cross platform password manager application was introduced br In System Settings individual menus have been rearranged and slightly redesigned for quick access to frequently used menus br Safari has been revamped including revamped reader mode now called Reader faster page load times a new start page and a brand new unified menu these had previously been exclusive to compact mode on iOS iPadOS br macOS Sequoia will release with the second iteration of Game Porting Toolkit a Windows API compatibility layer derived from Wine and Crossover allowing developers to more easily port Windows games to macOS br Apple Intelligence will be available on all Apple silicon Macs and will include artificial intelligence features such as a revamped and redesigned Siri Writing Tools with which selected text can be proofread or have its tone changed Smart Reply in Mail to quickly draft email responses and a system wide integration with ChatGPT br Window tiling is now automatically suggested by macOS when dragging a window to the edges of the display similar to Microsoft Windows Window snapping feature br The Tips app is now a standalone app has a new UI and includes many new tips for individual apps such as Freeform Pages Keynote Notes and Apple TV br br br
