Mac OS X Public Beta internally code named Kodiak was the first publicly available version of Apple Computer s Mac OS X now named macOS operating system to feature the Aqua user interface It was released to the public on September for US Its release was significant as the first publicly available evidence of Apple s ability to ship the next generation Mac operating system after the Copland failure It allowed software developers and early adopters to test a preview of the upcoming operating system and develop software for it before its final release It is the only public version of Mac OS X to have a code name not based on a big cat until the release of Mavericks in The US version had a build number of H and the international version had build number E br br br Successor OS br The Public Beta succeeded Mac OS X Server the first public release of Apple s new NeXT OPENSTEP based operating system which used a variant of the classic Mac OS s Platinum user interface look and feel The Public Beta introduced the Aqua user interface to the world Fundamental user interface changes were revealed with respect to fonts the Dock the menu bar with an Apple logo at the center that was later repositioned to the left of the menu bar and made an active interface element System icons were much larger and more detailed and new interface eye candy was prevalent br br br Technical changes br The beta s arrival marked some fundamental technical changes most courtesy of an open source Darwin core including two features that Mac users and developers had been anticipating for almost a decade preemptive multitasking and protected memory To illustrate the benefits of the latter at the MacWorld Expo in June Apple CEO Steve Jobs demonstrated Bomb app a test application intended to crash br br br Native software br The Public Beta included many of the standard programs bundled with macOS for decades to come such as TextEdit Preview Mail QuickTime Player and Terminal Also included with the Public Beta but not in any subsequent versions of Mac OS X were a simple MP player iTunes had not yet been introduced Sketch a basic vector drawing program demonstrating features of Quartz and HTMLEdit a WYSIWYG HTML editor inherited from WebObjects br Native shrinkware applications were few and far between Early adopters had to turn to open source or shareware alternatives giving rise to an active homebrew software community around the new operating system Many programs in use on early Mac OS X systems were inherited from OPENSTEP or Rhapsody developer releases e g OmniWeb or Fire or were simple wrapper apps that provided a graphical interface to a command line Unix program br The poor state of the Carbon API contrasted with the relative maturity of Cocoa gave rise to an anti Carbon bias among Mac OS X users br br br Expiration br The Mac OS X Public Beta was expired on May approximately two months after the release of Mac OS X the completed version of the operating system released in March As a result it will not run on later PowerPC based Macintosh computers released after early nor on current Macintosh hardware which uses the x or ARM processor architectures Using the Mac OS X Public Beta on compatible equipment today requires setting the hardware clock to a date prior to the expiration date br The expiration date forced users to purchase a copy of the final release rather than continuing to use the Public Beta as well as reassured industry observers skeptical after the Copland and Rhapsody failures that Apple would actually release a next generation operating system this time Owners of the Public Beta version were entitled to a discount on the price of the first full version of Mac OS X Only the Aqua GUI and related components of the Public Beta were subject to expiry the underlying Darwin command line based OS continued to function br br br
