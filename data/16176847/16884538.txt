Moblin short for mobile Linux is a discontinued open source operating system and application stack for Mobile Internet Devices MIDs netbooks nettops and embedded devices br Moblin was built around the Intel Atom processor All builds of Moblin were designed to minimize boot times and power consumption as Moblin was a netbook and MID centric operating system The netbook desktop version of Moblin supported other chipsets based on the SSSE instruction set such as the Core and some Celeron processors br OEM support was scarce but hit an all time high in when Acer replaced Linpus Linux with
