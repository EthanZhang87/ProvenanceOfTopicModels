MenuetOS is an operating system with a monolithic preemptive real time kernel written in FASM assembly language The system also includes video drivers It runs on bit and bit x architecture computers Its author is Ville M Turjanmaa It has a graphical desktop games and networking abilities TCP IP stack One distinctive feature is that it fits on one MB floppy disk br br br History br br br bit br MenuetOS was originally written for bit x architectures and released under the GPL only license thus many of its applications are distributed under the GPL br br br bit br The bit MenuetOS often referred to as Menuet remains a platform for learning bit assembly language programming The bit Menuet is distributed without charge for personal and educational use only but without the source code and the license includes a clause that prohibits disassembly br Multi core support was added on Feb br br br Features br MenuetOS development has focused on fast simple efficient implementation MenuetOS has networking abilities and a working TCP IP stack Most of the networking code is written by Mike Hibbett br The main focus of Menuet has been on making an environment for easy assembly programming but it is still possible to run software written in high level programming languages on the assembler core The biggest single effort towards high level language support is Jarek Pelczar s work in porting C libraries to Menuet br The GUI at version supports display resolutions up to x million colours with window transparency The OS has support for several classes of USB peripherals MenuetOS ships with the shareware versions of Quake and Doom br For disk access MenuetOS supports the FAT file system Write support is only possible to USB connected devices br As of version MenuetOS
