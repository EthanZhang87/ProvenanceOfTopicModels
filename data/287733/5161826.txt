MetaType also stylized as METATYPE is a tool for creating Type fonts using MetaPost developed by the Polish JNS team Bogus aw Jackowski Janusz Marian Nowacki and Piotr Strzelczyk br Since Metafont cannot produce outline fonts vector based a new tool was needed to help creating such fonts primarily for use with TeX although the OpenType versions of the fonts might be used in any other program It is less powerful than Metafont since no pens can be used only filled paths but it still allows creation of parametric fonts br Most important fonts produced with MetaType are Latin Modern Latin Modern Math TeX Gyre Antykwa Toru ska Antykwa P tawskiego Kurier and Iwona br br br br Yannis Haralambous Fonts and Encodings O Reilly ISBN pages br br br Notes br br br External links br Homepage of Janusz M Nowacki with fonts produced with MetaType br ftp with the program MetaType Windows br MetaType for Unices
