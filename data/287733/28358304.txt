Ubuntu is an OpenType based font family designed to be a modern humanist style typeface by London based type foundry Dalton Maag with funding by Canonical Ltd The font was under development for nearly nine months with only a limited initial release through a beta program until September It was then that it became the new default font of the Ubuntu operating system in Ubuntu Its designers include Vincent Connare creator of the Comic Sans and Trebuchet MS fonts br The Ubuntu font family is licensed under the Ubuntu Font License br br br History and features br The font was first introduced in October with the release of Ubuntu in four versions Regular Italic Bold and Bold Italic in English With the release of Ubuntu in April additional fonts and expanded language coverage were introduced The final development is intended to include a total of thirteen fonts consisting of br br Ubuntu in Regular Italic Bold and Bold Italic br Ubuntu Monospace in Regular Italic Bold and Bold Italic br Ubuntu Light in Regular and Italic br Ubuntu Medium in Regular and Italic br Ubuntu Condensed in Regular only br The monospace version used in terminals was initially planned to ship with Ubuntu However it was delayed and instead shipped with Ubuntu as the default system monospace font br The font is fully Unicode compliant and contains Latin A and B extended character sets Greek polytonic and Cyrillic extended In addition it has become the first native operating system font to include the Indian rupee sign The font has been designed primarily for use on screen displays and its spacing and kerning is optimized for body copy sizes br br br Usage br The Ubuntu Font Family is the default font for the current and development releases of the Ubuntu operating system and is used for the Ubuntu project branding br Ubuntu has been included in the Google Fonts directory making it easily available for web typography and as of April it is included for use in Google Docs br Ubuntu Monospace was prominently used in the video game Transistor br Ubuntu bold italic is also used in the bitcoin logotype alongside the bitcoin symbol br Ubuntu sans is the font used in the GIF Maker by the popular GIF sharing app Tenor the font has become very popular in internet meme culture being used commonly to caption GIFs and images br TV Tropes has used Ubuntu in its logo since br br br
