Graphics Layout Engine GLE is a graphics scripting language designed for creating publication quality graphs plots diagrams figures and slides GLE supports various graph types such as function plots histograms bar graphs scatter plots contour lines color maps and surface plots through a simple but flexible set of graphing commands More complex output can be created by relying on GLE s scripting language which is full featured with subroutines variables and logic control GLE relies on LaTeX for text output and supports mathematical formula in graphs and figures GLE s output formats include EPS PS PDF JPEG and PNG br The GLE software dates back to the early s and it is still under active development Currently GLE development is hosted on GitHub br br br Platforms br GLE is available for all major operating systems including Microsoft Windows Mac OS X and Unix It is included in Linux distributions such as Debian Fedora and SUSE br br br License br GLE is licensed under the BSD license QGLE the GLE user interface is licensed under the GPL license br br br See also br Asymptote br Gnuplot br Tikz br List of graphing software br br br br br br External links br Official website
